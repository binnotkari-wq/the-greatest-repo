3. Activer le boot graphique (Plymouth)

Pour que vous ne voyiez plus les messages texte au démarrage, vous devez modifier les paramètres du chargeur de démarrage (GRUB) :

    Ouvrez le fichier de configuration : sudo nano /etc/default/grub

    Cherchez la ligne : GRUB_CMDLINE_LINUX_DEFAULT="quiet"

    Modifiez-la pour ajouter "splash" : GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"

    Enregistrez (Ctrl+O, Entrée) et quittez (Ctrl+X).

    Mettez à jour GRUB : sudo update-grub
